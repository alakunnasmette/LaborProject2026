# LABOR Arbeidsmarktintegratie - Career Assessment Platform

## Overview

LABOR is a comprehensive career assessment platform designed to help you evaluate and understand your career development potential. The application guides you through five evidence-based assessments that measure personality traits, career values, interests, workplace culture preferences, and ideal job characteristics. Your answers are automatically saved, and detailed results can be exported to professional Excel reports.

### Key Functionality
- **5 Integrated Assessment Tools** - complete personality and career evaluations in one program
- **Real-Time Session Tracking** - resume your assessments anytime without losing progress
- **Automatic Data Management** - all client information and responses stored securely
- **Professional Excel Reports** - export comprehensive results for analysis and record-keeping
- **Intuitive Client Management** - easily add, edit, and manage multiple client profiles
- **No Internet Required** - all processing happens locally on your computer

---

## How the Program Works

### Client Management System

When you launch LABOR, you'll see the **Client Management Dashboard** where you can:

1. **View All Clients** - See a complete list of all clients who have started assessments
2. **Search Clients** - Quickly find a specific client by name
3. **Add New Clients** - Create a new client profile with personal information
4. **Edit Client Information** - Update details at any time
5. **Access Client Dashboards** - Select a client to view their assessment status and history

Each client has their own secure profile with:
- Personal contact information
- Assessment progress tracking
- Session history
- Generated reports

### Assessment Flow

For each client, you can launch assessments in this order:

1. **Assessment Questionnaire** - Complete all 5 phases of career assessments
2. **Prognosis Questionnaire** - Additional analysis (if enabled)
3. **View Results** - Review completed assessments and generated reports

## What You'll Assess

### Phase 1.1: Big Five Personality Assessment (50 questions)
This fundamental personality assessment measures five key dimensions:
- **Introversion vs. Extraversion** - How outgoing and social you are
- **Agreeableness** - How cooperative and compassionate you are
- **Conscientiousness** - How organized and disciplined you are
- **Neuroticism vs. Emotional Stability** - Your emotional resilience
- **Openness to Experience** - Your creativity and intellectual curiosity

*Duration: ~10-15 minutes | Question format: 5-point scale (Strongly Disagree to Strongly Agree)*

### Phase 2.0: Career Anchors Assessment (30 questions)
Identifies your core career values and motivations. Discover which of these resonates most with you:
- **Advancement** - Climbing the ladder and achieving success
- **Security** - Stable, reliable work environments
- **Independence** - Autonomy and freedom in how you work
- **Balance** - Priorities beyond just career
- **Challenge** - Working on exciting, complex problems

*Duration: ~5-10 minutes | Question format: Multiple-choice selections*

### Phase 2.1: Career Clusters (Career Interests)
Explores your interests across different career categories to identify fields that match your preferences.

### Phase 2.2: Organizational Culture Fit
Assesses your preferences for different work environments and organizational cultures to find the best fit.

### Phase 2.3: Job Characteristics Models
Evaluates what specific job characteristics matter most to you in your ideal role.

## Getting Started: Installation & Setup

### Prerequisites
- Windows computer (Windows 7 or newer)
- Administrator access to install the application

### Installation

**1. Download the Application**
   - Download the `LABOR-Assessment-Platform.exe` file
   - Save it to a convenient location (Desktop or Program Files recommended)

**2. Run the Installer**
   - Double-click the `.exe` file
   - Follow the installation wizard
   - Accept the terms and choose your installation directory
   - The application will automatically set up all required components

**Note: if you get a promt saying "Windows Protected Your Computer", simply press "More information" and then click "Run Anyway"**
   Windows often flags applications that are downloaded from external sources, so this is not uncommon.

**3. Launch LABOR**
   - After installation completes, click "Launch LABOR" or find the application in your Start Menu
   - The main client management dashboard will open

**That's it!** No additional setup is needed.

## Using the Program: Complete Workflow

### Starting Your First Assessment

**1. Open LABOR** - Launch the application from your desktop or Start Menu

**2. Create a New Client Profile**
   - Click the **"+ Nieuwe Klant"** (New Client) button in the top right
   - Fill in the client's personal information:
     - **Naam** (Name) - required
     - **Geboortedatum** (Date of Birth)
     - **Email**
     - **Telefoonnummer** (Phone Number)
     - **Adres** (Address)
     - **Opleidingen** (Qualifications/Education)
     - **Land van herkomst** (Country of Origin)
     - **Leef situatie** (Living Situation)
     - **Toelichting** (Additional Notes - optional)
   - Click **"Opslaan"** (Save)

**3. Open a Client's Dashboard**
   - Find the client in the list and double-click their name
   - The client dashboard displays:
     - All stored client information
     - **📋 Assessment Vragenlijst** button to start assessments
     - **🔮 Prognosis Vragenlijst** button for additional analysis
     - **📊 Bekijk Resultaten** button to view completed reports
   - You can edit their information at any time with the **✏️ Edit** button

**4. Start an Assessment**
   - Click **"📋 Assessment Vragenlijst"** (Assessment Questionnaire)
   - The system automatically checks for incomplete sessions:
     - **If previously started**: You'll be asked if you want to resume the last incomplete phase
     - **If first time**: You'll begin at Phase 1.1
   - Click **"Ja"** (Yes) to resume or the assessment starts fresh

### During an Assessment

**Navigation and Progress:**
- Answer each question carefully and honestly on the left side of the screen
- Click **"⬅️ Terug"** (Back) to review previous answers
- Click **"➡️ Volgende"** (Next) to proceed to the next question or phase
- **Progress is automatically saved** - you can close and reopen anytime
- There's no time limit - take as long as needed

**Question Formats Vary by Phase:**
- **Phase 1.1**: 5-point scale (Strongly Disagree to Strongly Agree)
- **Phase 2.0**: Radio button selections from multiple statements
- **Phase 2.1**: Checkbox selections
- **Phase 2.2**: Likert scale ratings (1-5)
- **Phase 2.3**: Free-text responses

**After Completing All Phases:**
- The final screen shows a completion confirmation
- You're returned to the client dashboard
- Results are now available for export

### Exporting and Viewing Results

**After completing assessments:**

1. Open the client's dashboard
2. Click **"📊 Bekijk Resultaten"** (View Results)
3. An Excel file is generated containing:
   - **Complete assessment scores** for all five phases
   - **Statistical analysis** and breakdowns
   - **Career recommendations** based on responses
   - **Detailed interpretations** of each domain

**Export Location:**
- Excel files are saved in the client's data folder automatically
- You can share these reports with clients or supervisors

### Managing Multiple Clients

**Search Function:**
- Use the search bar at the top to quickly find clients by name
- Results update as you type

**Edit Client Information:**
- Open a client's dashboard and click the **✏️Bewerken** (Edit) button
- Update any information in the form
- Click **"Opslaan"** to save changes

**Deleting Client Information**

- When opening a client's dashboard you will see a red button with the text **"🗑️Verwijderen"** (Delete)
- Pressing this button will trigger a prompt asking you for confirmation if you'd want to delete this client
- Deleted Clients **CANNOT** be recovered

**Client Data:**
- All client information and assessment responses are stored locally
- Session data persists indefinitely
- Client folders are organized in the application's data directory

## The Five Assessment Phases - Detailed Functionality

### Phase 1.1: Big Five Personality Assessment (50 questions)

**What it measures:**
This comprehensive personality assessment evaluates five core dimensions of personality:

- **Extraversion vs. Introversion** - Your sociability, assertiveness, and energy levels in social situations
- **Agreeableness** - Your tendency to be compassionate, cooperative, and considerate of others
- **Conscientiousness** - Your organization, discipline, responsibility, and attention to detail
- **Emotional Stability vs. Neuroticism** - Your resilience under stress and emotional well-being
- **Openness to Experience** - Your creativity, intellectual curiosity, and preference for novelty

**How it works:**
- You'll see 50 statements one at a time
- For each statement, rate your agreement on a 5-point scale:
  - 1 = Strongly Disagree
  - 2 = Disagree
  - 3 = Neutral
  - 4 = Agree
  - 5 = Strongly Agree
- Your responses are immediately scored
- Results show your percentile rank on each dimension

**Time required:** 10-15 minutes

**Output:**
- Score breakdown for each of the 5 personality dimensions
- Percentile rankings compared to general population
- Interpretation of what scores mean for career fit

---

### Phase 2.0: Career Anchors Assessment (30 questions)

**What it measures:**
Identifies the core values and motivations that drive your career decisions. Career anchors represent what is most important to you in your professional life.

**The Five Career Anchors:**

1. **Technical/Functional Competence** - Expertise in your field; you want to be known as highly skilled
2. **Managerial Competence** - Leadership and climbing the organizational ladder
3. **Security/Stability** - Job security, predictability, and reliable employment
4. **Entrepreneurship/Independence** - Starting your own business or maximum autonomy
5. **Service/Dedication to a Cause** - Making a difference and helping others

**How it works:**
- You'll see 30 statements, each paired with a career anchor
- For each statement, indicate whether it describes you using radio buttons
- The system tracks which anchor appears most frequently in your selections
- Your strongest anchor(s) are identified

**Time required:** 5-10 minutes

**Output:**
- Primary career anchor(s) identified
- Secondary motivations
- Recommendations for suitable career paths
- Analysis of fit with current role

---

### Phase 2.1: Career Interest Clusters

**What it measures:**
Explores which types of work and career fields align with your interests and preferences.

**Career Categories Assessed:**
- Art & Creative Industries
- Business & Finance
- Education & Training
- Engineering & Technology
- Healthcare & Social Services
- Law & Public Service
- Sales & Customer Service
- Trades & Technical Fields

**How it works:**
- Checkboxes allow you to select multiple interests
- Rate your interest level in different career statements
- The system clusters similar responses
- Identifies which career fields best match your interests

**Output:**
- Ranked list of career fields that match your interests
- Interest profile comparison
- Specific job titles that may suit you
- Industry recommendations

---

### Phase 2.2: Organizational Culture Fit

**What it measures:**
Determines which types of work environments and organizational cultures you prefer and perform best in.

**Culture Dimensions:**
- **Hierarchical vs. Flat** - Structured command chains or collaborative teamwork
- **Innovative vs. Stable** - Rapid change and experimentation or established practices
- **Task-Focused vs. People-Focused** - Results-oriented or relationship-oriented
- **Competitive vs. Collaborative** - Individual achievement or team success
- **Formal vs. Casual** - Professional standards or relaxed atmosphere

**How it works:**
- You'll rate statements on a 5-point Likert scale (1-5)
- Each rating reflects your preference for different cultural aspects
- Multiple questions assess each dimension
- Results show your cultural profile

**Time required:** 8-12 minutes

**Output:**
- Culture fit profile mapped to organizational types
- Best and worst environments for you
- Recommendations for company/role selection
- Insights into potential job satisfaction factors

---

### Phase 2.3: Job Characteristics Models

**What it measures:**
Identifies which specific job features and work characteristics are most important to your career satisfaction.

**Key Job Characteristics Evaluated:**
- **Variety** - Diverse tasks and responsibilities
- **Autonomy** - Freedom to make decisions and work independently
- **Feedback** - Clear information about job performance
- **Significance** - Impact of your work on others
- **Identity** - Clear completion of identifiable tasks
- **Learning Opportunities** - Chance to develop new skills
- **Work-Life Balance** - Time for personal life
- **Compensation** - Salary and benefits
- **Job Security** - Stability and predictability
- **Social Interaction** - Collaboration and teamwork

**How it works:**
- You'll provide free-text responses or rate importance
- Describe what matters most to you in a job
- Indicate what you want to avoid or improve
- The system analyzes patterns in your responses

**Time required:** 10-15 minutes

**Output:**
- Ranked list of job characteristics you value most
- Analysis of current role vs. preferences
- Specific recommendations for role modifications
- Career development suggestions

## How Data is Stored and Managed

### Automatic Data Saving
- **Every answer is saved instantly** - There's no "Save" button because the program saves automatically
- If the program closes unexpectedly, your progress is preserved
- When you return, you can resume exactly where you left off
- **No internet connection required** - Everything happens locally on your computer

### Session Persistence
- Client profiles and assessment responses are stored in JSON format
- Data is organized by client in individual folders
- You can resume assessments days, weeks, or months later
- All historical data is retained

### Results Export
- Click **"📊 Bekijk Resultaten"** to generate an Excel report
- Reports include all scores, analyses, and recommendations
- Excel files are automatically created from assessment data
- Multiple exports can be generated without losing original data

## Troubleshooting

### The Program Won't Start
- Try restarting your computer
- Ensure you have administrator privileges
- Delete and reinstall if startup issues persist
- Check that Windows is up to date

### Application Crashes During Assessment
- Close the program and reopen it
- The incomplete assessment will be available to resume
- Try completing one phase at a time if you experience crashes
- Contact your administrator if crashes continue

### Can't Find or Open a Client
- Use the search function to locate the client by name
- Check that the client name is spelled correctly
- Try refreshing the client list (close and reopen the program)
- Contact your system administrator

### Excel Reports Not Generating
- Ensure you have sufficient disk space
- Verify that you have write permissions to the data folder
- Try a different export location
- Ensure Microsoft Excel or compatible software is installed

### Missing Assessment Data
- Check that all 5 phases have been completed
- Resume and verify all answers were saved
- Try exporting again
- Contact your administrator if data appears lost

## Best Practices for Using LABOR

### For Administrators/Assessors

1. **Complete Your Own Assessments First** - Become familiar with all phases before administering
2. **Ensure Honest Responses** - Brief clients that assessment accuracy depends on truthful answers
3. **Provide Adequate Time** - Don't rush clients through assessments; plan 45-60 minutes minimum
4. **Keep Client Data Secure** - Regularly back up client files and exported reports
5. **Review Results Together** - Discuss findings with clients to maximize utility

### For Clients Taking Assessments

1. **Be Honest** - Assessment accuracy depends on truthful, genuine responses
2. **Take Your Time** - There's no time pressure; think carefully about each question
3. **Complete All Phases** - Finish all 5 phases for the most comprehensive picture
4. **Ask Questions** - It's okay to ask for clarification on any question
5. **Discuss Results** - Review your career recommendations with your assessor or mentor
